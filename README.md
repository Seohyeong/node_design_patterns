# Node.js Design Patterns Study Notes

This repository is my study space for working through exercises and writing recap notes from *Node.js Design Patterns (4th Edition)* by Mario Casciaro and Luciano Mammino.

My background is mostly in machine learning with PyTorch and Python, but the recent AI boom has pulled me deeper into fullstack work and building AI applications end to end.

At some point I realized I could keep duct taping JavaScript together with vibe coding and autocomplete, or I could actually learn Node.js properly. So this repo is me choosing the second option. 

## Source Material

- Book: *Node.js Design Patterns (4th Edition)*
- [Node.js Design Patterns: Fourth Edition is out!](https://loige.co/nodejs-design-patterns-fourth-edition/)

## Running Examples

Exercise files uses modern Node.js with ESM-style imports.

To run an exercise:

```bash
node ch3/3.1.js
node ch3/3.2.js
...
```

## Notes

- These are study exercises, so some files include experiments, rough edges, and extra logging.
- The goal here is learning and understanding, not polished production code.
